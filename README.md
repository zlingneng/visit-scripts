# 脚本文件备份工具使用说明

## 功能说明
本工具可以自动将指定的脚本文件复制到备份目录中，按照药店和医院分类整理。

## 使用方法

### 方法一：使用文本文件管理文件路径（推荐）
脚本优先从文本文件加载路径：
1. 编辑 `药店文件路径.txt` 文件，每行添加一个药店相关文件的完整路径
2. 编辑 `医院文件路径.txt` 文件，每行添加一个医院相关文件的完整路径
3. 支持多种路径格式：
   - 普通路径：`/Users/a000/拜访规划/file.py`
   - 带注释的路径：`/Users/a000/拜访规划/file.py 这是注释`
   - 带引号的路径：`"/Users/a000/拜访规划/file.py"` 或 `'/Users/a000/拜访规划/file.py'`
   - 带引号和注释的路径：`"/Users/a000/拜访规划/file.py" 这是注释`
4. 运行脚本：`python3 脚本文件备份工具.py`

### 方法二：使用脚本中的注释区域
如果文本文件未提供路径，脚本会尝试从注释区域提取文件路径：
1. 在 `{PHARMACY_FILES_START}` 和 `{PHARMACY_FILES_END}` 之间添加药店文件路径
2. 在 `{HOSPITAL_FILES_START}` 和 `{HOSPITAL_FILES_END}` 之间添加医院文件路径
3. 路径可以使用以下格式：
   - 带引号：`'/Users/a000/拜访规划/文件名.py'`
   - 不带引号但带注释符号：`# /Users/a000/拜访规划/文件名.py`

### 方法三：硬编码默认路径
如果以上两种方法都未提供路径，脚本将使用内置的默认路径列表。

## 注意事项
- 脚本会自动创建药店和医院两个备份目录
- 脚本会自动复制自身到备份目录根目录
- 支持的文件类型：.py, .txt, .csv, .json
